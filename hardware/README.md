# Hardware

PCB design and schematic files for the custom flight controller board.

## Structure

```
hardware/
├── kicad/           # KiCad project files
│   ├── schematic/   # Schematic sheets
│   ├── pcb/         # PCB layout
│   ├── lib/         # Custom component libraries
│   └── gerber/      # Manufacturing outputs
├── bom/             # Bill of materials
├── mechanical/      # Mechanical drawings / enclosure
└── datasheets/      # Component datasheets (gitignored, large files)
```

## Tools

- [KiCad](https://www.kicad.org/) — Schematic capture & PCB layout
