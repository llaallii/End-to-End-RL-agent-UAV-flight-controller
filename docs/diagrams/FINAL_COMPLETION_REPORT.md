## Lessons Learned

### What Worked Well
1. **Mermaid.js Selection:** Excellent for version-controlled diagrams
2. **Incremental Creation:** Building diagrams in priority order
3. **Dual Format Rendering:** PNG for docs, SVG for presentations
4. **Integration Strategy:** Embedding during creation phase
5. **Color Coding:** Consistent color schemes across diagram types

### Challenges Overcome
1. **Rendering Complexity:** Simplified overly complex diagrams (D7.1)
2. **Layout Optimization:** Adjusted dimensions for readability (3500x2500)
3. **Subgraph Errors:** Learned Mermaid subgraph limitations
4. **File Organization:** Created category-based folder structure

### Technical Insights
1. **Mermaid Limits:** Max ~50 nodes per diagram for stable rendering
2. **Flowchart vs Graph:** Flowchart syntax more reliable than graph TB
3. **Styling Best Practices:** classDef after all node definitions
4. **Export Settings:** Dark theme + transparent background optimal
