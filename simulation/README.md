# Simulation

ROS 2 Jazzy + Gazebo Harmonic simulation environment for SITL UAV development.

## Structure

```
simulation/
├── gazebo/          # Gazebo world files, models, plugins
│   ├── worlds/
│   ├── models/
│   └── plugins/
├── ros2_ws/         # ROS 2 workspace
│   └── src/         # ROS 2 packages
├── launch/          # Launch files
├── config/          # Simulation parameters
└── scripts/         # Utility scripts
```
